Zoroak

Previous Next


As the final project in AP Physics, we created Zoroak, a remote soil analyzer. In a group of five, we used Raspberry Pi and Python to create an autonomous soil arm that extracts soil samples and analyzes the light spectra, temperature, and soil density. OpenCV was used to calibrate position and depth. Motion is adjusted in real time.
